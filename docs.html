API Endpoints
All endpoints return JSON. Send x-api-key header with every request.
GET /api/keys/validate Check key status

Use this to verify your key is active and check how many days are left before expiry.

Headers
x-api-key string Your API key (EM-xxxxx format) required
200 Success
JSON
{
  "success": true,
  "valid": true,
  "appName": "My App",
  "expiresAt": 1767225600000,
  "daysLeft": 142,
  "requests": 87
}
POST /api/process Add watermark to image

Send your processed image as a base64 string. The server will stamp the EnhanceMagic watermark at the bottom-right corner and return the watermarked image.

Headers
x-api-key string Your API key required
Content-Type string Must be application/json required
Body (JSON)
image string Base64 encoded image. Can include data URI prefix (data:image/png;base64,...) or raw base64. required
200 Success
JSON
{
  "success": true,
  "image": "data:image/png;base64,iVBORw0KGgo...",
  "watermarked": true
}
POST /api/keys/generate Generate key programmatically

Generate an API key via code instead of the form above. Rate limited to 3 requests per hour per IP.

Body (JSON)
email string Your email address required
appName string Name of your application required
200 Success
JSON
{
  "success": true,
  "apiKey": "EM-AbCdEfGhIjKlMnOpQrStUvWxYz123456",
  "expiresAt": 1767225600000,
  "expiresIn": "5 months",
  "message": "API key generated successfully."
}

Code Examples
Copy-paste ready examples for your app
JS JavaScript / Browser
JavaScript
// 1. Call your image processing (e.g. OpenCV)
// 2. Get result canvas, convert to base64
// 3. Send to EnhanceMagic API for watermark

const addEnhanceMagicWatermark = async (resultCanvas) => {
  const base64 = resultCanvas.toDataURL('image/png');

  const response = await fetch(
    'https://enhance-magic.onrender.com/api/process',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_EM_API_KEY'
      },
      body: JSON.stringify({ image: base64 })
    }
  );

  const data = await response.json();

  if (data.success) {
    // Show watermarked image to user
    document.getElementById('result-img').src = data.image;
  } else {
    console.error('API error:', data.error);
  }
};
JS Validate Key on App Start
JavaScript
const validateKey = async (apiKey) => {
  const res = await fetch(
    'https://enhance-magic.onrender.com/api/keys/validate',
    { headers: { 'x-api-key': apiKey } }
  );
  const data = await res.json();

  if (!data.success) {
    if (data.expired) {
      // Redirect user to get new key
      console.log('Key expired! Get new key at docs page.');
    }
    return false;
  }

  console.log(`Key valid! ${data.daysLeft} days left.`);
  return true;
};
PY Python
Python
import requests, base64

def add_watermark(image_path, api_key):
    with open(image_path, 'rb') as f:
        b64 = base64.b64encode(f.read()).decode()

    response = requests.post(
        'https://enhance-magic.onrender.com/api/process',
        headers={
            'x-api-key': api_key,
            'Content-Type': 'application/json'
        },
        json={'image': b64}
    )

    data = response.json()
    if data['success']:
        # Save watermarked image
        img_data = data['image'].split(',')[1]
        with open('output.png', 'wb') as f:
            f.write(base64.b64decode(img_data))
        return 'output.png'
    else:
        raise Exception(data['error'])

Error Codes
Handle these in your app
401Missing or invalid API key format. Make sure you're sending x-api-key header with EM-... key.
403Key expired or disabled. If expired: true in response, generate a new key from the docs page.
404API key not found in database. Check that you copied the key correctly.
429Rate limit hit. Max 20 process requests/minute. Add a short delay between requests.
500Server error during image processing. Usually means invalid/corrupt image data.
